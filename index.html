Frequently Asked Question

Q1: How does javascript work?

Ans:javascript is single thread high level,dynamic,scripting language.It runs inside the web browser.When the JavaScript code is loaded in the browser, JavaScript engine(V8) will execute each line of the file from first to last.JavaScript V8 engine will parse the code, convert it into machine code and then execute it.

The V8 does two major things;1.Heap memory allocation 2.Call stack execution context

Q2: Difference between local storage and session storage

Ans:Local storage: stores data with no expiration date/permanent.local storage data is more secure.local storage capacity is 5/10 Mb.(vary browser to browser ).

Session storage:stores data for a session.session storage is cleared when the page session ended.it page loads faster.session storage capacity is 5 Mb.
